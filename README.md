# U.S. States Search Engine (C++)

## Description
Console-based application that parses all 50 U.S. states from a text file into custom State objects.

## Features
- Search by state abbreviation
- Multi-match search by year of statehood
- Object-oriented design using classes

## Concepts Used
- Vectors
- File I/O
- Classes & encapsulation
- Loops and conditionals
